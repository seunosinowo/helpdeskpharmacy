# Helpdesk Pharmacy 🏥

Helpdesk Pharmacy is a modern, responsive, and user-friendly pharmacy website built with **Next.js 14+** and **Tailwind CSS**. It features a professional design with interactive elements, animations, and a focus on customer service and accessibility.

## ✨ Features

- **Modern UI/UX**: Clean, professional design with a focus on ease of use and accessibility.
- **Responsive Design**: Fully functional across all devices (mobile, tablet, desktop).
- **Interactive Elements**: Uses **Framer Motion** for smooth animations and scroll reveals.
- **Service Management**: Detailed showcase of pharmaceutical services (Prescription, Wellness, Consultations).
- **Contact System**: Built-in contact form for customer inquiries.
- **Optimized Performance**: Leverages Next.js App Router and optimized assets for fast loading.

## 📁 Project Structure

- `src/app`: Next.js App Router pages, layouts, and global styles.
- `src/components`: Reusable UI components, layout sections, and shadcn/ui components.
- `src/assets`: Images, logos, and static assets.
- `src/lib`: Utility functions and shared configuration.
- `src/hooks`: Custom React hooks for shared logic.
